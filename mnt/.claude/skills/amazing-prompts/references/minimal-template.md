# Minimal Tier Template

Use for quick queries, simple lookups, casual conversation.

## Template

```
[PERSONA]
Act like a [role]. Explain [topic] for [audience].

[CONSTRAINTS]
- Length: [limit]
- Format: [type]

[TASK]
[Your actual question or request]
```

## Example Usage

```
[PERSONA]
Act like a senior data analyst. Explain the difference between correlation and causation for a business manager.

[CONSTRAINTS]
- Length: under 100 words
- Format: plain prose

[TASK]
What is the key distinction I should understand?
```

## When to Use

- Simple factual questions
- Quick explanations
- Casual conversation
- Tasks with obvious correct answers
- Low stakes output
