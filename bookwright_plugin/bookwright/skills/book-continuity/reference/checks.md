# The consistency battery

What book-continuity checks, and how each splits between a script and a reader.

Counts, by the script. Opener present on every non exempt chapter, the `Definicija` band, the figure band, the two callout bands, a closing exercise. Anything a number can answer comes from `structure_scan.R`.

Spine reconciliation, script flags and a reader judges. The defined terms should match the chapter's key terms, the opener should name them, the exercises should drill them. The script can check whether the key terms appear in the opener and the exercises. Whether they are the right terms is a reader's call, handed to `critic-structure`.

Terminology, by the script against the concept ledger. A term defined twice, used before its introducing chapter, or called two different names for one concept.

Voice and arc, by readers. `critic-voice` judges whether the book sounds like one author, `critic-arc` judges whether it builds. These need reading the whole sequence and cannot be counted.

Bands. On the first run, measure the current counts across all chapters and propose bands from the distribution. The aim is symmetry drawn from the book you wrote, not arbitrary targets.
